# space_war_game_in_python

# Description: -
This is a simple Space War game made in Python and PyGame.

# Local Set-Up :-
    $ git clone https://github.com/nitinsaini1008/space_war_game_in_python.git
    $ pip3 pygame
    $ python3 space_war_game.py
